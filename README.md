# Rokid Glasses Landing Page

Landing page giới thiệu sản phẩm công nghệ Rokid Glasses, được xây dựng bằng React + Vite + Tailwind CSS. Website tập trung vào giao diện hiện đại, responsive, tối ưu hiệu năng, có form kết nối dữ liệu ra bên ngoài và nhiều chức năng mở rộng.

## Live Demo

https://rokid-landing-page.vercel.app/

## GitHub Repository

https://github.com/Kai2464/rokid-landing-page

## Công nghệ sử dụng

- React
- Vite
- Tailwind CSS
- Motion for React
- Lucide React
- Web3Forms
- Vercel

## Chức năng chính

- Hero Section giới thiệu sản phẩm Rokid Glasses
- Section tính năng nổi bật
- Section ứng dụng thực tế dạng scrollytelling
- Section thông số kỹ thuật
- Form đăng ký nhận tin qua Web3Forms
- Giao diện Dark Mode
- Scroll progress bar
- Hiệu ứng Parallax nhẹ
- Skeleton Loading
- Micro-interactions
- Mini e-commerce: lưu sản phẩm yêu thích, giỏ hàng và sản phẩm đã xem
- Form gửi yêu cầu tư vấn sản phẩm qua Web3Forms
- Chatbot tự động ở góc màn hình, không cần API

## Điểm cộng đã thực hiện

- Kiểm tra tính hợp lệ dữ liệu form trước khi gửi
- Gửi dữ liệu người dùng về Web3Forms thật
- Hiển thị thông báo khi người dùng click hoặc scroll
- Phát triển chế độ Dark Mode
- Scroll Animation
- Skeleton Loading
- Micro-interactions
- Scrollytelling kết hợp Parallax
- Mini e-commerce
- Chatbot tư vấn sản phẩm tự động
